Análista de requerimientos

Definió requerimientos técnicos, diseñó el sistema de monitoreo ambiental, optimizó su ubicación, documentó especificaciones clave y validó la integración y consistencia de los datos en el sistema IoT.

Arquitecto de sistemas

Diseñó e implementó el sistema de alumbrado público, utilizando sensores LDR para regular la intensidad luminosa según las condiciones ambientales, optimizando el consumo energético. Documentó especificaciones técnicas y colaboró en la integración del sistema al IoT, asegurando su funcionalidad y escalabilidad.

Jefa de proyecto

Coordinó todas las etapas del proyecto, supervisando la integración de los subsistemas en un sistema IoT funcional. Diseñó el sistema de monitoreo de nivel de residuos utilizando sensores ultrasónicos y optimizó la gestión energética mediante soluciones técnicas, como la administración de voltaje externo. Garantizó la conectividad con la base de datos y el dashboard, asegurando una integración eficiente y resultados alineados con los objetivos del proyecto.

Desarrollador de Software

Diseñó y programó el código para los microcontroladores así como la conexión de estos con la API, permitiendo el envío de datos en tiempo real a una base de datos en la nube, a su vez generó consultas SQL para gestionar y analizar la información recolectada. Además, colaboró en la visualización de los datos en un dashboard, asegurando una integración eficiente entre hardware, software y análisis de datos. Su trabajo fue clave para automatizar el sistema y garantizar su funcionalidad en el entorno urbano.

Desarrollador de Software

Implementó y calibró el sistema de drenaje inteligente utilizando el sensor de lluvia YL-83 y un servomotor para controlar las alcantarillas, optimizando su respuesta ante lluvias moderadas e intensas. Diseñó e integró los componentes de hardware del sistema IoT, asegurando la comunicación eficiente con el software y la base de datos. Su enfoque en la interconexión de sensores y actuadores fue esencial para garantizar un funcionamiento robusto y escalable del proyecto.
