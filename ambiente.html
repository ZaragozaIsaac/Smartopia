¿Por qué es importante?

El sistema fue diseñado para monitorear variables como temperatura y humedad, permitiendo adaptar políticas públicas al cambio climático y mejorar la calidad de vida de los ciudadanos.

¿Cómo funciona?

El sensor DHT11 mide la temperatura y la humedad en tiempo real, mostrando los datos en una pantalla LCD y almacenándolos en una base de datos para análisis histórico.

Impacto urbano

Proporciona datos que ayudan a diseñar estrategias para mitigar los efectos del cambio climático y mejorar el confort en áreas urbanas

Proyección a futuro

Se planea expandir el sistema a múltiples puntos urbanos para crear un mapa climático detallado y enviar alertas automáticas en caso de condiciones extremas.
